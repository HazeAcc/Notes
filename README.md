**关于行数统计：**

在 VS Code 中使用 <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>F</kbd> 开启全局搜索，在正则表达式模式下匹配：`b*[^:b#/]+.*`

自动忽略空行。字数统计大概为行数的3至4倍。
